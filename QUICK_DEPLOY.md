# Quick Deploy Guide - Get Online in 5 Minutes

## Fastest Way: Vercel (Recommended)

### Prerequisites
- GitHub account (free)
- Your code is ready

### Steps

1. **Create GitHub Repository** (2 min)
   - Go to [github.com/new](https://github.com/new)
   - Name: `albums-landing-page`
   - Click "Create repository"

2. **Push Your Code** (2 min)
   ```bash
   # In your project folder
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin https://github.com/YOUR_USERNAME/albums-landing-page.git
   git branch -M main
   git push -u origin main
   ```

3. **Deploy to Vercel** (1 min)
   - Go to [vercel.com](https://vercel.com)
   - Sign up with GitHub
   - Click "Add New Project"
   - Select your repository
   - Click "Deploy"
   - Wait 1 minute
   - **Done!** Your site is live!

### Your URL
You'll get: `albums-landing-page.vercel.app`
